Posted on 2017-03-09 | In Python学习笔记 | Visitors

python单行注释符号(#)

井号(#)常被用作单行注释符号，在代码中使用#时，它右边的任何数据都会被忽略，当做是注释。

#号右边的内容在执行的时候是不会被输出的。
