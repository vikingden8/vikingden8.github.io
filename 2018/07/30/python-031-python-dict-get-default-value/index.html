同样地，上面的实现方式应该也是可以的，但是，我们仍可以用一种简单的方式来实现。也就是，Python中的字典有一个 get() 方法，当在键不存在时，可以提供一个默认值返回。

当在使用 get() 方法时，首先会去检查当前的键是否在字典中存在。若存在，则返回键所对应的值；若不存在，则返回 get() 方法中提供的默认值。
